# Data

The CSV files are **not** included in this repository.

Download them from the Kaggle dataset
[Credit Card Approval Prediction](https://www.kaggle.com/datasets/rikdifos/credit-card-approval-prediction)
and place both files in this folder:

- `application_record.csv`
- `credit_record.csv`

The notebook expects them at `data/application_record.csv` and `data/credit_record.csv`.
